Ilse's Cheat Sheet

HTML Basics

Tag/Attribute Name Description
<div> Division Defines a division, used as a container for elements. Good for class or id attribute.
<h1> Heading Used for the largest headings. Followed by h2 to h6.
<br> Line break Produces line break in text (return).
<hr> Horizontal rule Produces a horizontal line between elements.
<ul> Unordered list Creates a list using bullet points. Used together with <li>.
<ol> Ordered list Creates a numbered list. Used together with <li>.
<img> Image Embeds an image into the HTML document. Needs to have src="" defined. Self-closing.
<a> Anchor Embeds a hyperlink into the HTML document. Needs to have href="" defined.

HTML Tables

Tag/Attribute Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The data inside the table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.
colspan Column Span Defines how many columns a td element should span.
rowspan Row Span Defines how many rows a td element should span.

CSS Selectors & Specificity

Selector Example Description
.class .intro TSelects all elements with class="intro"
class1.class2 .name1.name2 Selects all elements with both name1 and name2 set within its class atribute. Also known as chaining.
.class1 p .name1 p Selects all <p> elements that are descendants of an element with class="name1".
#id #title Selects the element with id="title".
element p Selects all <p> elements.
element.class p.intro Selects all <p> elements with class="intro".
element, element div, p Selects all <div> and <p> elements.
element element div p Selects all <p> elements inside <div> elements.
Class overrides tag & id overrides class. <!important overrides all other selectors.
